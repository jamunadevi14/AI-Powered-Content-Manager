# AI-Powered Content Manager

An interactive, client-side web application built to streamline content creation using artificial intelligence. This application integrates directly with the Google Gemini API to generate customized text dynamically based on user prompts.

## 🚀 Features

* **Smart Content Generator:** Generates custom blog posts given a specific topic and optional target keywords.
* **Text Summarizer:** Condenses long, complex articles or text fragments into concise, easy-to-read summaries.
* **Tone Transformer:** Rewrites text blocks to alter their emotional or professional tone to match specific audience requirements.
* **Secure API Key Handling:** Built with a client-side configuration panel that processes user API tokens securely straight through the UI context rather than hardcoding credentials.

## 🛠️ Technology Stack

* **Frontend Structure:** HTML5
* **Styling & Layout:** CSS3 (Sidebar layout with active tab management)
* **Application Logic:** JavaScript (Asynchronous Fetch API / ES6+)
* **AI Core Engine:** Google Gemini API 

## 📦 How to Use

1. Clone or download this repository containing `index.html`, `style.css`, and `script.js`.
2. Open `index.html` directly in any modern web browser.
3. Obtain a free API key from Google AI Studio.
4. Paste your key into the **Gemini API Key** field in the configuration sidebar.
5. Select your tool choice, input your text or topic prompts, and click **Generate Content**.
